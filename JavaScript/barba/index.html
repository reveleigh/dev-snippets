How it works

Barba.js works by hijacking links, fetching the new page via fetch, and swapping the content of data-barba="container" inside data-barba="wrapper".

<body data-barba="wrapper">
  <!-- Persistent Header -->
  <main data-barba="container">
     <!-- Dynamic Content -->
  </main>
</body>
